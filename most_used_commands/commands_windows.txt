1. net user: Print all the users in windows
2. net accounts: Display information regarding password configurations and logon restrictions
3. net localgroup: Display information about the groups
4. lusrmgr.msc: This can be typed in run terminal to view users and group and manage them
5. dir: Display contents of a directory
6. hostname: View hostname information of system
7. systeminfo: View detailed information regarding the system
8. type: Read the contents of a document
9. rename old_name new name: Rename the file or directory from old to new one
